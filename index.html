Project Description

        
  • The main goal of this project is to create a visualization that shows useful data about how successful certain musical artists are on tour. We do this by displaying different information on a US Map and bar graphs (as of right now). We gathered this information from Wikipedia pages specific to the tours we are using (more about that later), and we created the visualization using d3 Javascript.
    •     
  • Our main motivation for this project comes from us being avid music fans. We really enjoy music, and going to concerts, but it seems that a lot of major artists tend to avoid coming to St. Louis. We were curious as to why artists avoid St. Louis, and some cities, and tend to gravitate towards other cities. We thought that if we could gather information about different artist's performances, like the attendance, revenue, etc, we can start to determine which cities artists have the most success in.

US Map Color Coding Explanation

This information corresponds to the artist chosen in the selector on the left of the page.

Top 5 Performance with respect to artist performances overall. If a state has both a top 5 revenue value and a top 5 attendance value it will be purple.
Top 5 Performance in terms of only revenue. If a state has a top 5 revenue value but not a top 5 attendance value it will be red.
Top 5 Performance in terms of only attendance. If a state has a top 5 attendance value but not a top 5 revenue value it will be blue.
Performed in State. If the artist performed in a state it will be dark grey

State Pie Chart Explanation

This information corresponds to the artist chosen in the selector on the left of the page. If you click on a state in the map, one of two things will happen:


  1. You are alerted with the message "There were no concerts in this state :("
  2. A pie chart appears below the map

The first option occurs if the artist did not perform in the state you clicked on.

The second option shows the amount of revenue that the artist made at each venue in the state. You can hover over the slices to see the venue, revenue, and % revenue.
